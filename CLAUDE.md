## What this repo is

An internship report written in [Typst](https://typst.app/). The final output is `report.pdf`, produced by overlaying a separately-prepared header PDF onto the compiled body.

## Build

Compile the body:
```bash
typst compile src/report.typ out/report-body.pdf
```

Merge header + body into the final `report.pdf` (requires `pypdf`):
```bash
python out/concat.py
```

The header (`out/report-header.pdf`) comes from `src/report_header.docx` converted to PDF externally — it is not regenerated by these commands.

## Structure

- `src/report.typ` — entry point; sets all global formatting (fonts, heading styles, page numbering, inline code styling) and `#include`s every section in order.
- `src/sections/intro.typ` — front matter (acknowledgements, abstract, TOC, figures, abbreviations); uses Roman page numbering (`I`).
- `src/sections/1_introduction.typ` … `8_conclusion.typ` — numbered body chapters; Arabic page numbering restarts at 1.
- `src/sections/4_presentation.typ` — currently commented out in `report.typ`.
- `src/components/timeline.typ` — reusable Typst component(s).
- `src/references.bib` — BibTeX bibliography, rendered with IEEE style at end of document.
- `out/concat.py` — PDF merge script; overlays header pages onto matching body pages using `pypdf`.

## Heading numbering scheme

Level 1 headings use `I.1.1.` format; levels 2–3 use `1.1.1.`; level 4+ use `a.1.` (offset from depth 3). This is defined in `src/report.typ` via the `header-numbering` function.
