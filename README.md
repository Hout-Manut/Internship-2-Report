# A Repository for my Internship 2 Report

This report is being made with [Typst](https://typst.app/).

## Output
Please check the `out` folder for the compiled report file.

## Compilation
To manually compile the report. First install the (Typst compiler)[https://typst.app/open-source/#download].

Then run 
```bash
typst compile src/report.typ out/report-body.pdf
```
